XMLHttpRequest ()

XMLHttpRequest 对象是浏览器提供的一个API，用来向服务器发送请求并解析服务器响应

方法 描述
.open() 准备启动一个AJAX请求
.send() 发送AJAX请求
.setRequestHeader() 设置请求头部信息
.getResponseHeader() 获得响应头部信息
.getAllResponseHeaders() 获得一个包含所有头部信息的长字符串
.abort() 取消异步请求
属性 描述
.responseText 包含响应主体返回文本
.status 响应的HTTP状态
.statusText HTTP状态的说明
.readyState 表示“请求”/“响应”过程的当前活动阶段

XMLHttpRequest 方法

XHR.open( method, url[, async] )

method : 请求方式，如 GET，POST 等
url : 请求地址
async : 是否异步执行，可选，默认为 true

XHR.send( [body] )

用于发送 http 请求，如果请求方法为 GET 或 HEAD，参数应设为 NULL

XHR.setRequestHeader(header, value)

此方法的调用必须在 open() 后，send() 前
header : 属性名
value : 属性值
对同一请求头重复赋值，会生成一个合并多个值后的请求头

XHR.getResponseHeader(name)

name : 属性名
返回相对应的属性值

XHR.getAllResponseHeaders()

返回所有响应头

XHR.abort()

如果请求已发出，此方法将终止该请求，当一个请求被终止，它的 readyState 属性将被置为 0

属性

responseText

一个由 XMLHttpRequest 返回的字符串，当请求状态 readyState 值为 4 ( DONE ) 且 status 值为 200 时，responseText 时后端返回的全部数据

status

返回 XMLHttpRequest 响应中的数字状态码，在请求完成前，其值为 0

值为 1XX : 代表临时的响应信息
值为 2XX : 服务器成功接收了客户端请求
值为 3XX : 重定向
值为 4XX : 客户端错误，如请求了不存在的页面
值为 5XX : 服务器错误，不能完成请求

readyState

返回一个 XMLHttpRequest 代理当前所处的状态

值为 0 : 代理被创建，但尚未调用 open() 方法
值为 1 : open() 方法已经被调用
值为 2 : send() 方法已经被调用，并且头部和状态已经可获得
值为 3 : 下载中，responseText 属性已经包含部分数据
值为 4 : 下载已完成

一个 Ajax 请求

GET

let xhr = new XMLHttpRequest()
xhr.onreadystatechange = () => {
    if (xhr.readystate == 4) {
        if (xhr.status >= 200 && xhr.status < 300) {
            alert(xhr.responseText)
        } else {
            alert("Error : " + xhr.status)
        }
    }
}
xhr.open("get", "api.php", true)
xhr.send(null)

POST

let xhr = new XMLHttpRequest();
xhr.open('post', '02.post.php');
xhr.setRequestHeader("Content-type", "application/x-www-form-urlencoded");
xhr.onreadystatechange = function () {
    if (xhr.status == 200) {
        if (xhr.status >= 200 && xhr.status < 300) {
            alert(xhr.responseText);
        } else {
            alert("Error : " + xhr.status)
        }
    }
};
xhr.send('name=role&age=18');
